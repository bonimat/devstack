Ambiente DESTACK: APACHE2-PHP7.2-PHPUNIT8-XDEBUG

Ciao,
l'ambiente di sviluppo ha caricato il web server Apache.
La cartella html/ che contiene questo file index.html é nella path del progetto docker.
Per cambiarla deve essere rilanciato l'ambiente dopo aver impostato la variabile globale $DEVSTACK_WWWROOT col comando:

export DEVSTACK_WWWROOT="PATH"

oppure

export DEVSTACK_WWWROOT=(echo $PHPP)

nel caso sia definita una variabile di sistema che contiene il percorso dei progetti php.
